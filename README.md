
# Saving App

## server
To run server
- create mySql db
- `cd server`
- open .env file and put new db info in
    -  `DB_SRVER` `DB_NAME` `DB_USER` `DB_PWSD` 
- `npm start`
server will start on port 3000 or 4000

- `flutter pub get`
